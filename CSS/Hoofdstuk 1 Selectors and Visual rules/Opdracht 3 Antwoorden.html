Type selector: Type selectors past elementen aan per tagnaam. Hierin worden dan alle elementen binnen het HTML sheet aangepast aan wat er in de CSS sheet mee word gegeven. Voordeel hiervan is dat dezelfde soort elementen in 1 keer kan aanpassen zonder het indivudueel te doen per zelfde soort tag. Voorbeeld: HTML:

Voordeel hiervan is dat je zelfde tags apart van mekaar kan stylen. Je kan hiermee specifiek 1 tag binnen je div id aanpassen en de rest laten zoals het is. Voorbeeld: HTML:

Class selector: Bij class selector pas je de CSS in de HTML tag zelf. Of bij class selector zet je de id in de tag waarbij je de class in de tag zelf doet. Voordeel hiervan is dat je tags die hetzelfde zijn afzonderlijk van mekaar kunt bewerken in css zonder ze allemaal te nemen. Hierin worden de tags met de classes binnen in het tag alleen meegenomen in de CSS aanpassing, terwijl de andere tags zonder de class binnen de tag niet worden meegenomen in de CSS aanpassing.
